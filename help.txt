MANAGE PLUGINS
  qwer plugin-add <name> <git-url>     Add git repo as plugin
  qwer plugin-list                     List installed plugins
  qwer plugin-remove <name>            Remove plugin and package versions
  qwer plugin-update <name>            Update plugin
  qwer plugin-update --all             Update all plugins


MANAGE PACKAGES
  qwer install <name> <version>        Install a specific version of a package
  qwer uninstall <name> <version>      Remove a specific version of a package
  qwer which <name>                    Display version set or being used for package
  qwer where <name> <version>          Display install path for an installed version
  qwer list <name>                     List installed versions of a package
  qwer list-all <name>                 List all versions of a package


UTILS
  qwer reshim <name> <version>         Recreate shims for version of a package
